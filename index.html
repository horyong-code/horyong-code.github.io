📌 Summary

2차 프로젝트 주제인 MES에 맞춰서 원두를 가공, 포장하여 판매하는 MES를 구축해보았습니다.
1차 프로젝트와는 다르게 SPRING을 메인으로 사용하여 진행했으며 기본 기능에 충실하여 구현 완료 후 다양한 부가 기능을 구현해보려고 노력해보았습니다.

* 맡은 파트
  • DB 관리
    • 제가 맡은 품질과 재고 테이블을 구성하고, 전체 ERD 작성 및 DB 관리를 수행하였습니다.
  • 품질 관리 페이지 구현
    • 품질 관리 현황과 불량 현황의 목록을 출력하는 페이지를 구현하였습니다. 자재 입고, 생산, 반품 처리 시 데이터를 받아 품질을 검사하는 로직을 구현하였습니다.
    품질 검사 후 정상은 재고로, 불량은 불량 정보를 입력 후 불량 정보로 보내는 로직을 구현하였습니다.
  • 재고 관리 페이지 구현
    • 자재와 생산의 재고 목록을 출력하는 페이지를 구현하였습니다. 재고량 변경과 창고 이동 로직을 구현하였습니다.

1차 프로젝트와는 다르게 SPRING을 메인으로 사용하여 진행했으며 기본 기능에 충실하여 구현 완료 후 다양한 부가 기능을 구현해보려고 노력해보았습니다.

* 맡은 파트
  • DB 관리
    • 전체 DB 구성, ERD 작성 및 프로젝트 기간동안 전반적인 DB 관리를 하였습니다.
  • 마이페이지 구현
    • 회원 정보 확인과 전화번호, 이메일 중복 확인을 통한 회원 정보 수정을 할 수 있도록 구현하였습니다.
    간략한 주문 내역 조회를 할 수 있도록 목록 출력을 하였습니다.
    비밀번호 입력 후 확인을 통한 회원 정보 수정 및 탈퇴를 구현하였습니다.
  • 고객센터 페이지 구현
    • 기본적인 게시판 형태로 고객센터 페이지를 구현하였습니다.
  • 관리자 페이지 구현
    • 주문 내역, 회원 목록, 푸드트럭 목록, 상품 목록을 한 페이지에서 전체적으로 조회할 수 있도록 하였습니다.
    푸드트럭 등록과 그 푸드트럭에 따른 상품을 등록할 수 있는 페이지를 구현하였습니다.
    회원을 강제로 삭제할 수 있으며 활성화 컬럼을 등록하여 DB에서는 일정 기간동안 보유하고 있도록 하였습니다.
